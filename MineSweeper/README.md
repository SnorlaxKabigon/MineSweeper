# マインスイーパ (MineSweeper) 取扱説明書

このゲームは、通常のマインスイーパに「コイン」「スキン」「称号」などのやり込み要素を追加したWebブラウザゲームです。

## 🎮 ゲームの基本ルール

基本ルールは通常のマインスイーパと同じですが、**ライフ制**を採用しています。

1.  **目的**: 地雷以外のすべてのマスを開けること。
2.  **操作**:
    *   **Dig (掘る)**: マスをクリックして開けます。数字は周囲8マスの地雷の数を示します。
    *   **Flag (旗)**: 地雷だと思うマスに旗を立てます（誤操作防止）。
3.  **ライフ制（重要）**:
    *   このゲームでは**2回まで地雷を踏んでもゲームオーバーになりません**。
    *   **3回目**の地雷を踏むとゲームオーバーになります。
4.  **勝利条件**: 地雷以外のすべてのマスを開けること。

## 🕹️ 操作方法

*   **左クリック**: マスを開ける (Digモード時) / 旗を立てる (Flagモード時)
*   **モード切替**: 画面下部の「Dig」「Flag」ボタンで切り替え可能。
*   **顔アイコン**: クリックするとゲームをリセット（再挑戦）します。

## 🏆 難易度と報酬

難易度によって盤面の広さと地雷の数が異なります。クリアタイムに応じてコインを獲得できます。

| 難易度 | サイズ | 地雷数 | 獲得コイン (Time) |
| :--- | :--- | :--- | :--- |
| **Easy** | 9x9 | 10 | 10枚 (30秒以内) / 5枚 (60秒以内) / 1枚 (その他) |
| **Normal** | 16x16 | 40 | 20枚 (90秒以内) / 10枚 (180秒以内) / 5枚 (その他) |
| **Hard** | 32x32 | 199 | 30枚 (600秒以内) / 15枚 (900秒以内) / 10枚 (その他) |

## ✨ 特徴的な機能

### 1. ショップ (Skin Shop)
集めたコインを使って、ゲーム中の顔アイコン（スキン）を変更できます。
*   **アクセス**: プロフィール画面の「Skin Shop」ボタンから。
*   **ラインナップ**:
    *   🙂 Default (0コイン)
    *   🤓 Chee (1コイン)
    *   😴 Sleep (10コイン)
    *   👽 Alien (200コイン)
    *   🤖 Robot (500コイン)
    *   😈 Devil (1000コイン)

### 2. 称号と実績 (Achievements)
プレイ状況に応じて「称号」やボーナスコインが獲得できます。
*   **プレイ回数ボーナス**: 10回, 20回, 50回, 100回プレイでコイン獲得。
*   **称号 (Titles)**:
    *   クリア回数に応じてランクアップ (Rookie -> Master)。
    *   地雷をたくさん踏むと手に入る隠し称号も…？
*   **称号の変更**: プロフィール画面の「Achievements」から獲得した称号を装備できます。装備した称号はプロフィールに表示されます。

### 3. コンティニュー機能 (Recovery)
ゲームオーバーになっても、**20コイン**を支払うことでゲームを再開できます。
*   再開時はライフが1つ（あと1回ミスで終了）の状態になります。

### 4. ランキング
難易度ごとにクリアタイムのランキングが表示されます。上位を目指してタイムアタックに挑戦しましょう。

## 👤 アカウント機能

*   **ゲストプレイ**: 登録なしでも遊べますが、データは保存されません。
*   **ユーザー登録**: 名前とパスワードを登録すると、コインや称号、スキンのデータを保存できます。
